# Session Log — 2026-04-19 (Phase 6)

## Work done

Phase 6 — Post-Session View — fully implemented.

### Decisions made
- Strava credentials in gitignored `lib/config/strava_config.dart`; user fills in their own client ID + secret. Template at `strava_config.template.dart` committed for reference.
- OAuth: temporary localhost HttpServer on OS-assigned port captures callback code; system browser opened via url_launcher. User sees browser once ever; subsequent exports refresh token silently.
- Strava "Authorization Callback Domain" must be set to `localhost` in the Strava app settings.
- Export format: TCX (XML) — simpler to generate than FIT, accepted by Strava. One trackpoint per second with watts + HR.
- TSS computed from Normalized Power (30-second rolling 4th-power mean). NP uses an O(n) sliding window approach.
- Save and Export are independent actions; Export saves first if not already saved.
- "Skip" navigates to Planner without saving. "Save" and "Export" both navigate to Calendar after completing.
- RPE must be selected before Save/Export buttons are enabled.
